# SwarggerJSON-API

**SwarggerJSON-API**为了减少频繁的从 swargger 复制粘贴到项目中，特此写了这个生成器

因为项目本身使用的 yapi，而且日常经常会使用到 api 的复制粘贴。所以特此写了该生成器。

## 安装和使用

### 1.安装依赖项

```js
yarn;
```

或

```js
npm install
```

### 2.生成 api.js

```js
yarn api
```
